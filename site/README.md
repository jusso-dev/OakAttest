# OakAttest Static Showcase

This directory contains the static showcase site for GitHub Pages.

The site uses the product screenshots generated by:

```bash
npm run screenshots:readme
```

The GitHub Pages workflow copies this directory into a temporary publish folder,
then copies `docs/screenshots/*.png` and the OakAttest logo assets from
`public/` into `assets/`.

To preview locally after assembling assets:

```bash
mkdir -p site/assets/screenshots
cp docs/screenshots/*.png site/assets/screenshots/
cp public/oak-attest-mark.png site/assets/
python3 -m http.server 8080 --directory site
```

Then open `http://localhost:8080`.
