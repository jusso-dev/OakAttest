Built for Australian IRAP assessment work

Evidence, scope, findings and artefacts in one defensible workspace.

OakAttest is an open-source assessment workspace for ASD-registered assessor firms and the client organisations they assess. It keeps the IRAP lifecycle structured from organisation setup through scoping, evidence, fieldwork, findings, certification and ongoing compliance maintenance.

View the product See deployment model

Why this matters

The Australian market does not need another generic compliance tracker.

IRAP work depends on boundaries, ISM applicability, evidence quality, client collaboration, residual risk, auditability and exportable records that can support an authorisation decision. OakAttest is shaped around those practical tasks, with self-hosted deployment as a first-class option.

IRAP lifecycle coverage

Built around the work assessors already have to perform.

Plan

Organisation and engagement setup

Create the assessor organisation, invite colleagues, define client contacts, assign roles and establish engagement scope.

Scope

Boundary and Cloud IRAP context

Model environments, data centres, services, inherited provider controls and system boundaries for AWS, Azure or Google Cloud workloads up to PROTECTED.

Assess

Control decisions and fieldwork

Record implementation statements, assessment methods, evidence quality, limitations, vulnerability imports and CVE or SBOM data.

Report

Findings, risk and certification

Link findings to ISM controls, record residual risk and prepare authorisation package material without losing the audit trail.

What it handles

A workspace for repeatable assessment operations.

Organisation controls Tenant roles, engagement-scoped client access, invitations, branding and IP allowlist administration.
Evidence lifecycle Requests, uploads, review decisions, generated artefact storage and append-only audit logs.
Control assessment ISM applicability, CAF record fields, evidence limitations and assessor decision history.
Security inputs Nessus, Rapid7, Qualys and generic CSV vulnerability scans, plus CVE and SBOM workflows.
Outputs SSP zip bundle, PDF material, Excel workbook, boundary image and certification records.

Deployment posture

Open source, self-hostable and careful about responsibility.

OakAttest is designed for deployments where assessment material, evidence and audit history need clear ownership. It does not claim that ASD certifies, endorses or approves a system. It helps assessor firms and clients collect and structure the artefacts that inform the relevant authorisation decision.
